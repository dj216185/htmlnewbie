1 · Your Very First Web Page 🏠

<!DOCTYPE html>
<html>
  <head>
    <title>My Cool Site</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
  </body>
</html>

Hello, world!

Did you know? The line <!DOCTYPE html> is like telling the browser, “Hey, I’m speaking modern HTML!” It isn’t shown on the page.

3 · Understanding the Style Tag and Adding Styles 🎨

In HTML, you can style your webpage in three main ways:

  • Inline Styles: Styles added directly to an HTML element via the style attribute.
  • Internal Styles: Styles included within the <style> tag in the <head> section of the document.
  • External Styles: Styles applied via an external CSS file linked using the <link> tag.

Here’s how they work:

Inline Style Example:

<p style="color: red; font-size: 20px;">This is a red text!</p>

This is a red text!

Internal Style Example:

<style>
  p { color: blue; font-size: 18px; }
</style>
<p>This is a blue text!</p>

This is a blue text!

External Style Example:

<link rel="stylesheet" href="styles.css">

4 · The Flow of an HTML Page 🔄

The flow of an HTML page follows this general order:

  • 1. : Declares the document type and version of HTML.
  • 2. : The root element that contains all HTML content.
  • 3. : Contains meta information, links to stylesheets, scripts, etc.
  • 4. : Contains the visible content of the webpage like headings, paragraphs, images, etc.

7 · Fantastic Tables 🍫

Tables are like spreadsheets inside a web page.

<table border="1">
  <caption>Chocolate Scores</caption>
  <tr>
    <th>Brand</th>
    <th>Yum Score</th>
  </tr>
  <tr><td>Dairy Wow</td><td>9</td></tr>
  <tr><td>Dark Blast</td><td>8</td></tr>
</table>
Chocolate Scores
BrandYum Score
Dairy Wow9
Dark Blast8
Did you know? You can add <thead>, <tbody> and <tfoot> parts to big tables so screen-readers understand them better.
